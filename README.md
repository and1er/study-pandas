# study-pandas

Study python's package Pandas for real world data analysis in Python.

## Requirements

* Python 3.7+;
* a python virtual environment with `requirements.txt` installed:

    ```bash
    python3 -m venv ./venv
    source ./venv/bin/activate
    pip install -r requirements.txt
    ```
